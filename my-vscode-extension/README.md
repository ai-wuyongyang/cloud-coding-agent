# Minimal VS Code Extension Demo

This is a tiny VS Code extension demo with an Activity Bar icon and a sidebar dialog.

It contributes:

- `Minimal Demo: Say Hello`
- A `Minimal Demo` icon in the Activity Bar
- A `Demo Dialog` webview in the sidebar

When the extension runs, click the `Minimal Demo` icon in the left Activity Bar to open the sidebar dialog.

## Run In VS Code

1. Open this folder in VS Code.
2. Press `F5`.
3. VS Code opens a new window titled `Extension Development Host`.
4. In that new window, click the `Minimal Demo` icon in the left Activity Bar.
5. Type a message in `Demo Dialog` and click `发送`.

You can also open the Command Palette with `Cmd+Shift+P` and run `Minimal Demo: Say Hello`.
